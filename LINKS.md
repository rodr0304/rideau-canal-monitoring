## Notes
This project demonstrates an end-to-end IoT data pipeline using:
- Azure IoT Hub
- Azure Stream Analytics
- Azure Cosmos DB
- Azure Blob Storage
- Azure Static Web Apps

The system collects simulated telemetry data from IoT devices and processes it in real time.
